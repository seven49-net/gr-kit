Element mit fixer Breite

Elemente, welche eine fixe Breite benötigen, sollten mit einem Blockelement mit Klasse "grch-fixed-width" umschlossen sein. Damit wird das horizontale Scrollen bei kleinen Bildschirm-Breiten ermöglicht.

Icons

Datei Icons

Datei Icons werden initialisiert mit der Klasse "grch-file-icon".

  • PDF (grch-file-icon grch-pdf)
  • Word (grch-file-icon grch-word)
  • Excel (grch-file-icon grch-excel)
  • Powerpoint (grch-file-icon grch-powerpoint)
  • Datei (grch-file-icon grch-file)
  • Bild (grch-file-icon grch-image)
